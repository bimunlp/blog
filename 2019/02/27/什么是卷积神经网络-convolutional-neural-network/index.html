什么是卷积神经网络(convolutional neural network)深度学习？

这篇文章我们来了解卷积神经网络的概念。卷积神经网络，也称为 CNN或ConvNet，是一种人工神经网络，迄今为止最常用的计算机视觉领域神经网络。

虽然图像分析是CNNS最广泛应用的领域，不过这种神经网络也可以用于其他数据分析或数据分类任务。让我们具体来看一看吧．

什么是CNN？

从本质上讲，CNN也是一种人工神经网络，它拥有能够检测并分离模式(pattern)的能力。这种对模式的识别能力使CNN在图像分析中非常有用。

CNN并不简简单单只是一个人工神经网络，它与标准的多层感知机（MLP）有着明显的区别．

CNN有被称作卷积层(convolutional layer)的隐藏层，这些层是构成CNN网络的关键所在，即卷积神经网络．

CNN网络的隐藏层中包含卷积层。

CNN网络中还有很多非卷积层，　但卷积层是CNN网络的基础。

那么卷积层的作用是什么？

卷积层

与任何其他层一样，卷积层接收输入，以某种方式转换输入，然后将转换后的输入输出到下一层。卷积层的输入称为输入通道，输出称为输出通道。

数据在卷积层中发生的变换称为卷积运算(convolution operation)。这个是深度学习从业者经常使用的名词，不过从数学上讲，卷积层执行的卷积运算实际应该被叫做互相关(cross-correlations)。

我们稍后会对这种运算作介绍。现在，让我们首先了解一下卷积层所做的变换背后所蕴含的深层思想。

滤波器(filter)和卷积运算

如前所述，卷积神经网络能够检测图像的模式。

对于每个卷积层，我们需要指定图层包含的滤波器的数量 。这些过滤器用来检测图像的模式。

模式(pattern)

我们所说的滤波器能够检测模式到底指的是什么？一张图片中包含了很多的元素，不同的边缘，形状，纹理，物体等。这就是我们所说的模式。

  • 边缘
  • 形状
  • 纹理
  • 曲线
  • 物体
  • 颜色

滤波器在图像中可以检测到的一种模式是边缘，因此滤波器又被称为边缘检测器。

除了边缘，一些滤波器可能会检测边角。有的可以检测圆弧，有的可以检测方形。这些简单的几何滤波器就是我们在卷积神经网络前端所看到的。

网络越深，滤波器就越复杂。在后面的层中，滤波器可以检测特定的物体，如眼睛，耳朵，头发或毛发，羽毛，鳞片或喙，而不是只是边缘和简单的形状。

在更深的层中，滤波器能够检测更复杂的物体，如犬，猫，蜥蜴和鸟类的整体。

为了理解这些卷积层及其各自滤波器是如何工作的，让我们看一个例子。

滤波器（模式检测器）

假设我们有一个卷积神经网络，向它传递手写体数字的图像（如MNIST数据集中的），训练的目标是人网络识别这些书写体各自代表的数值1， 2，3等。

假设模型中的第一个隐藏层是卷积层。如前所述，在向模型添加卷积层时，我们还必须指定卷积层的滤波器个数。

滤波器的数量决定了输出通道的数量。

从技术上讲，滤波器可以被认为是一个相对较小的矩阵（ 张量），我们需要选择该矩阵具有的行数和列数，并将矩阵内的值用随机数初始化。

对于第一个卷积层，我们给它指定一个大小为3 x 3的过滤器。

卷积层

让我们看一下卷积操作的示例动画：

此动画展示了以非数字呈现的卷积过程。底部是一个蓝色的输入通道，卷积滤波器在底部的输入通道依次投影到绿色的输出通道：

  • 蓝色（底部） - 输入通道
  • 阴影（蓝色上面） - 3 x 3卷积滤波器
  • 绿色（顶部） - 输出通道

对于蓝色输入通道上的每个位置，3 x 3滤波器进行匹配计算，将蓝色输入通道的阴影部分映射到绿色输出通道的相应阴影部分。

输入通道被卷积层接收之后，滤波器在输入的每组3 x 3像素上＂滑动＂，直到它从整个图像每个3 x 3像素块滑过。

卷积操作

这种滑动被称为卷积(convolving)，所以正确的说法是这个滤波器将在输入的每个3 x 3像素块上进行卷积。

蓝色输入通道是用矩阵表示的MNIST数据集中的图像。此矩阵中的值是图像中的各个像素。这些图像是灰度图像，所以我们只有一个输入通道。

  • 灰度图像具有单个颜色通道
  • RGB图像具有三个颜色通道

将此输入传递到卷积层。

我们指定第一个卷积层只有一个滤波器，这个滤波器在输入的每个3 x 3像素块上进行卷积。当滤波器落在第一个3 x 3像素块上时，会计算并存储滤波器本身与输入的3 x 3像素块的点积。对于每个3 x 3像素块滤波器都会执行这个操作。

例如，我们计算滤波器与第一个3 x 3像素块的点积，然后将结果存储在输出通道中。接下来滤波器滑动到下一个3 x 3 像素块，计算点积，并将该值存储为输出通道中的下一个像素。

在此过滤器对整个输入进行卷积之后，会形成对输入的新表示，现在将其存储在输出通道中。此输出通道称为特征映射(feature map)。

这个绿色输出通道成为下一层输入通道的输入，然后重复滤波器在上层的步骤，形成新的输出通道。

这只是一个非常简单的例子，这样的滤波器被视为模式检测器。

模式识别器是在学习中出现的。

过去的计算机视觉科学家是通过手动建立滤波器（模式检测器）。其中一个例子是Sobel滤波器，它是一种边缘检测器。随着深度学习技术的发展，我们开始能够使用神经网络自动习得这些过滤器．
